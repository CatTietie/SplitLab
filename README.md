# SplitLab

## 概述
SplitLab 是一个特性开关与 A/B 实验平台，帮助产品团队自助完成灰度发布和实验分析。你可以在后台创建实验、分配流量（百分比或白名单），业务代码通过轻量 SDK 拉取配置并进行本地分流，同时上报转化事件。平台自动计算各实验组的转化率与置信度，支持实验暂停、全量发布及配置回滚，所有操作留痕可审计。

## 核心功能
- **实验管理**：可视化创建多层互斥实验，灵活控制流量。
- **SDK 集成**：Python SDK 本地缓存配置，毫秒级分流，后台轮询更新。
- **数据分析**：自动统计转化率，提供 Z 检验置信度评估。
- **特性开关**：独立开关管理，支持灰度放量。
- **权限与审计**：修改审批、历史记录与回滚。

## 技术栈
- 后端：FastAPI, PostgreSQL, Redis
- 前端：React
- SDK：Python（模拟业务服务）
- 部署：Docker Compose

## 快速开始
1. 启动所有服务：`docker-compose up -d`
2. 访问管理后台：`http://localhost:3000`
3. 查看 `docs/` 了解实验配置 Schema、SDK 接口协议和统计方法文档。

## 测试
提供 10 万用户分流偏差 <0.5%、热更新秒级生效、事件零丢失等自动化测试，运行 `pytest` 验证。
